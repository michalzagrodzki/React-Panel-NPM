## Test npm react package

### Install

Install using `yarn add react-test-panel`

### Usage 

Inside React app, import Panel to component 
`import { Panel } from 'jareact-test-panelwblia'`  

#### Panel props

| Name        | Description
| ----------- | -----------
| title       | string
| content     | string
| children    | any
| onCancel    | function
| onSubmit    | function
